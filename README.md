# ⚙️ DataMaq Automation
**Ecosistema de soluciones de IA, Infraestructura y Desarrollo Web.**

Bienvenido al repositorio central de **DataMaq Automation**. Aquí se agrupan las estrategias, configuraciones y activos digitales que impulsan la eficiencia operativa y la transformación digital de nuestra organización.

---

## 🚀 Áreas de Operación

### 🤖 Inteligencia Artificial & Agentes
Desarrollamos y mantenemos una capa de agentes autónomos orientados a la toma de decisiones, gestión de riesgos y automatización de procesos internos.
- **Pilar:** [Tenazas (Kimiclaw)](./TENAZAS.md) - CEO virtual y asistente operativo avanzado.
- **Stack:** Google Vertex AI, OpenClaw, Node.js y MCPs personalizados.

### 🌐 Ecosistema Web
Gestión de presencia digital y plataformas de servicios técnicos.
- **[wp-datamaq](https://github.com/datamaq-automation/wp-datamaq):** Plataforma central de servicios en AMBA basada en WordPress.
- **[legacy-datamaq](https://github.com/datamaq-automation/legacy-datamaq):** Plantilla y activos históricos migrados para alta disponibilidad.

### 🏗️ Infraestructura & Cloud
Mantenemos sistemas híbridos para garantizar la resiliencia y escalabilidad de los servicios.
- **Nube:** Google Cloud Platform (GCE) e infraestructuras VPS optimizadas (AlmaLinux/Debian).
- **Local:** Entornos de desarrollo nativos sincronizados (Ubuntu 24.04).
- **Red:** Conectividad privada segura mediante Tailscale.
